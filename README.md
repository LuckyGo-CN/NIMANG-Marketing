# 逆芒营销推广订单处理系统（团队实验工作台版）

## 启动

双击 `start_server.cmd`，或在当前目录运行：

```powershell
python server.py --port 8023
```

可通过环境变量 `NIMANG_ADMIN_KEY` 修改后台密钥，通过 `NIMANG_DB` 指定数据库文件。

## 角色流程

推广专员01、02、03：练习任务A → 任务安排确认A → 团队推广任务A → 任务安排确认B → 团队推广任务B → 绩效结果。

推广专员04：练习任务B → 近期趋势理解 → 等待加入团队 → 任务安排确认B → 团队推广任务B → 绩效结果。

04 的“等待加入团队”会在同队 01、02、03 均完成团队推广任务A后自动解除。页面刷新或重新登录不会丢失已完成节点、任务提交或问卷答案。

## 数据

数据保存在 `data/experiment.db`。订单与问卷均使用统一的 `participant_uid`，后台可分别导出：

- 全部订单操作记录
- 当前场次订单记录
- 指定团队订单记录
- 绩效汇总
- 内嵌问卷明细

CSV 使用 UTF-8 with BOM，便于直接用 Excel 打开。后台的任务重置会同步回退相应工作流节点和问卷记录。
